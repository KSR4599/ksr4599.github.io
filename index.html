Chapter 1: Scaling Core Commerce Services

Walmart Global Tech | Sunnyvale, CA

Senior Software Engineer

01/2022 - Present | Intern -> SDE-3 -> Senior Software Engineer

Java Spring Boot Microservices AKS Redis Kafka

I work on backend services behind Walmart's tier-0 IAM and session platforms. The goal has been consistent: keep critical user flows fast, stable, and cost-efficient even during global traffic spikes.

  • Developed high-throughput SSO and federated session services for Walmart Commerce Platform, enabling one-login journeys across web, mobile, and partner channels with sub-150 ms global latency.
  • Rebuilt the legacy session flow into a multi-tenant backend stack; migrated 1.2B+ sessions in three days, sustained ~5K TPS, and scaled to 25M+ active sessions.
  • Improved runtime efficiency for a 20K OPM workload by reducing footprint from 480 to 225 pods, producing roughly $920K yearly savings and 30-50% latency improvement on critical APIs.
  • Delivered passkey/FIDO2 backend rollout on AKS with Flagger canary releases and regression monitoring across 100M+ monthly logins, with no production incidents.
  • Built CoreID routing logic for ASDA migration, enabling phased store-by-store cutover for 1,200 stores and 150K associates without disrupting POS or workforce access.

NASA | Fairfax, VA

Software Development Engineer

08/2020 - 01/2022

REST APIs JSON/CSV Time-series Data Interoperability

At NASA, I developed backend APIs for the Heliophysics Application Programmer Interface (HAPI) so mission data could be queried through a consistent, machine-friendly interface.

  • Implemented HAPI-compliant REST endpoints that ingested mission telemetry and served normalized JSON/CSV time-series data with rich metadata for scientific analysis.
  • Built storage-agnostic data access services so clients could query mission datasets without handling file-layout differences, unlocking cross-mission analysis over 3,000+ datasets and 60+ mission groups.
  • Worked with heliophysics data-center teams to improve API compatibility and error handling, reducing custom ETL/parser effort in research workflows.
